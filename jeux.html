Temps : ∅
Joueurs : 2; 3; 4; 5

Le but : Il faut marquer le plus de points.

Le joueur dispose d'une limite de 5 secondes pour tirer au panier.
Si le joueur rate un panier ou ne tire pas avant la limite, celui-ci est éliminé.
Le joueur dispose de 3 vies.
Temps : ∅
Joueurs : 2; 3; 4; 5

Le but : il faut aller le plus loin possible.

En tapotant sur son écran, le joueur fait avancer un petit oiseau.
Le joueur reçoit un point pour chaque tuyau évité par l'oiseau.
Si l'oiseau touche un tuyau, le sol, ou encore le ciel, le joueur est éliminé.
Temps : ∅
Joueurs : 2; 3; 4; 5

Le But : il faut aller le plus loin possible.

Le joueur incarne un dinosaure courant à toute vitesse.
En tapotant sur son écran, il le fait sauter par-dessus les différents obstacles.
Si le dinosaure touche un obstacle, le joueur est éliminé.
Temps : ∅
Joueurs : 2; 3; 4; 5

Le But : il faut aller le plus loin possible.

Une suite de couleur est annoncée.
Le joueur dispose d'une limite de temps pour retranscrire à l'identique l'ordre d'apparition de chaque couleur.
Si le joueur se trompe ou ne joue pas avant la limite, celui-ci est éliminé.
Temps : ∅
Joueurs : 2; 3; 4; 5

Le But : Il faut aller le plus loin possible.

Une suite de cases noires et blanches défilent à toute vitesse.
Le joueur doit toucher les cases noires avant que celles-ci ne disparaissent.
Si une case noire disparaît avant que le joueur n'est appuyé dessus, celui-ci est éliminé. Si le joueur appuie sur une case blanche, celui-ci est éliminé.
Temps : ∅
Joueurs : 2

Le but : il faut faire en sorte que la balle dépasse 5 fois la raquette adverse.

Une balle se déplace en rebondissant sur les bords de l'écran.
Les joueurs commandent chacun une raquette, représentée par un trait vertical situé aux extrémités gauche et droite du terrain de jeu.
Temps : 3 minutes 30
Joueurs : 2; 3; 4; 5

Le but : Terminer le puzzle

Le joueur doit faire glisser les cases dans le bon ordre.
Le premier joueur à terminer son puzzle remporte la partie.
Si après le temps réglementaire aucun joueur n'a terminé son puzzle, c'est le joueur ayant la plus grande suite et le moins de déplacement qui gagne.
Temps : ∅
Joueurs : 2; 3; 4; 5

Le but : il faut construire la plus haute tour possible.

Le joueur dispose d'une limite de 10 secondes pour aligner chaque brique.
À chaque fois qu'elle n’est pas complètement alignée à la précédente, une partie de la plateforme disparaît.
